Captioning Server Running

This server supports three client types:

  1. Web Client: A browser-based interface for captioning audio.
  2. Translation Client: A browser-based interface for audio translation.
  3. Terminal Client: A Python-based command-line client (captioning_client.py).

Note: Only one client can be connected at a time.

Open Web Client Open Translation Client

To use the terminal client, run:

python captioning_client.py
